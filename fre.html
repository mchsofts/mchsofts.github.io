Key Features
🔎📄

Text Search

Find text inside thousands of files within seconds. Advanced options for precision.

  • Recursive folder scanning
  • Case sensitive & whole word
  • Regular expressions
  • File mask filtering (*.txt, *.xml, *.cs, etc.)
  • Real-time result listing
✍️🔄

Text Replacement

Safely replace text across multiple files with full preview & backup.

  • Replace all matches or selected only
  • Preview before replacement
  • Automatic file backup
  • Detailed replacement log
  • Multi-file batch operations
📂⚡

File Search & Batch Rename

Quickly locate files & rename hundreds of items in one go.

  • Search by name, extension, size, date modified
  • Add prefix / suffix to filenames
  • Sequential numbering & replace text
  • Recursive folder processing
  • Rename selected files only

📁 Supported file types

TXT, LOG, INI, XML, JSON, CSV, HTML, CSS, JS, PHP, C#, C++, VB, Markdown, and many more text-based formats.
